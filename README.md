A Flutter application that helps users organize their wardrobe and create stylish outfits.

## Features

- Add clothes with images
- Categorize wardrobe
- Search and filter clothing
- Build outfits
- Save outfits
- Edit/Delete outfits
- Dark theme

## Built With

- Flutter
- Dart
- SQLite (sqflite)
